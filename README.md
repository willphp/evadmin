## EvAdmin

>一鱼PHP框架CURD快速开发平台

### 环境要求

- PHP7.4.3到PHP8.2.x最新版
- PDO扩展(MySQL数据库)

### 技术构架

- WillPHP框架v4.3
- PearAdmin最新版（Layui2.8.6）

### 主要功能

1. 后台菜单管理(增删改查)
2. 账户管理(增删改查)
3. 角色管理(RBAC)
4. 系统字典，用户字典，如配性别选顶
5. 图库管理，上传，删除(编辑器和单图上传的图片都存入图库)
6. 数据库表优化，修复，备份，还原
7. 模型管理，可生成数据表，生成控制器，模型类，视图模板，菜单
8. 字段字典，可预自定义字段让模型生成数据表时选择。
